Macの自動化に
必要なすべて

シンプルなマウスクリックから複雑なAI駆動ワークフローまで。 RocketMouse AIがあなたにスーパーパワーを与えます。

ビジュアルブロックエディタ

ブロックをスナップするだけで自動化フローを構築。7種類のブロック形状、色分けされたカテゴリ、 ドラッグ&ドロップ、元に戻す/やり直し、ミニマップナビゲーション、自然言語表示。 プログラミング経験不要。

AIアシスタント

やりたいことを日本語で説明するだけ。AIがマクロを自動生成。 macOS 26+ではApple Intelligenceがデフォルト（無料・オンデバイス）。 OpenAI、Anthropic、Gemini、Groq、ローカルLLMにも対応。

ブラウザ自動化

Playwright + Chromeを内蔵。ナビゲーション、クリック、フォーム入力、データ抽出、 スクリプト実行 — すべてビジュアルブロックから。さらにビジュアル要素ピッカーでCSS/XPathセレクタをクリックで選択。

Excel & ファイル操作

Excelファイルをネイティブに読み書き。18種類のスプレッドシート操作に加え、ファイルシステムへの完全アクセス — コピー、移動、作成、アーカイブ。

画像認識 & OCR

Apple Visionフレームワークが画像認識、テンプレートマッチング、OCRテキスト抽出を実現。 画面上の何でも見つけます。

記録 & 再生

2つの強力な記録モード: 入力記録はあらゆるアプリのマウスクリックとキー入力をキャプチャ。 ブラウザ記録はChrome上のクリック、フォーム入力、ページ遷移を自動キャプチャ。 どちらも編集可能なブロックを生成し、ループや条件分岐で自由に拡張できます。

クロスプラットフォーム

.rmprojファイルはmacOSとWindows間で完全互換。 Macで作って、Windowsで実行 — その逆も。

v2.0 新機能 — Computer Use

AIが画面を見て
操作する

セレクタも座標も不要。「何が見えるか」を言葉で伝えるだけ — AI Visionが画面をキャプチャし、 要素を見つけて操作します。あらゆるアプリを最も自然な方法で自動化。

👁

AIクリック

「送信ボタンをクリック」— AIが画面をキャプチャし、Vision LLMで要素を見つけ、 正確にクリック。画面端でも2パスリファインメントで高精度。

⏳

AIスマート待機

「ローディングが消えるまで待つ」— AIが画面を監視し、 条件が満たされたことを判断。固定待機も脆いセレクタも不要。

📖

AI OCR

「合計金額を読み取って」— AIが画面上のテキストを読み取り変数に格納。 どんなアプリ、どんな言語、どんなフォントでも対応。

✅

AI検証

「確認ダイアログが表示されている？」— AIが画面状態を確認し true/falseを返却。条件分岐フローに最適。

🚀

AIオートパイロット

「システム設定を開いてダークモードを有効にして」— AIが完全自律制御。 Anthropic Computer Use API（Claude Sonnet 4.6 / Opus 4.6）搭載。 クリック・入力・スクロール・キー操作をネイティブPC操作で自動実行。 Anthropic APIキーが必要。

🛡

セルフヒーリング

画像認識やOCRが失敗した場合、AI Visionが自然言語理解で 自動リトライ。UIが変わってもマクロが止まりません。

ハイブリッド実行

定型処理は高速・確実・無料。
AI は判断が必要な箇所だけ。

定型処理は macOS ネイティブ API で高速・確実・無料。UI 変動がある箇所だけ AI が視覚的に判断。 コストと信頼性の最適バランスを 1 本のマクロで実現できるのは RocketMouse AI だけ。
