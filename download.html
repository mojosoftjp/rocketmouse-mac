Double-click the DMG to mount it, then drag RocketMouse AI to your Applications folder.

Open RocketMouse AI from Applications. macOS will prompt you for the required permissions (see below).

Required Permissions

RocketMouse AI needs these macOS permissions to automate your system. You'll be prompted on first use.

⚙️
Accessibility

Required for automated mouse clicks, keyboard input, and window management.
System Settings → Privacy & Security → Accessibility

📷
Screen Recording

Required for screen capture, OCR, image recognition, and template matching.
System Settings → Privacy & Security → Screen Recording

⌨️
Input Monitoring

Required for macro recording (capturing mouse and keyboard events).
System Settings → Privacy & Security → Input Monitoring

📁
Files & Folders

Required for file operations and Excel file access. Grant when prompted.
