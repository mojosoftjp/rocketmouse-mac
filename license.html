The trial license provides full access to all features for 15 days from the date of first launch. No registration or payment is required. After the trial period expires, a Personal or Business license is required to continue using the software.

Personal License

A perpetual license for one individual user on one Mac. The license is bound to the hardware identifier of the registered Mac. Transfer to a different machine requires contacting support. The license key format is: RM-P-HHHH-XXXX-XXXX-CCCC

Business License

A perpetual license for organizations. Can be activated on unlimited devices. Cross-platform compatible — the same key works on both macOS and Windows. The license key format is: RM-B-****-XXXX-XXXX-CCCC

Privacy Policy

Data Collection

RocketMouse AI does not collect, transmit, or store any personal data, usage analytics, telemetry, or crash reports. The software operates entirely locally on your Mac.

AI Features

When using Apple Intelligence (macOS 26+), all AI processing happens entirely on your device — no data is sent to any server. When using cloud AI providers (OpenAI, Anthropic, Google, Groq, or custom endpoint), your prompts are sent directly to your chosen provider using your own API key. MojoSoft does not proxy, log, or have access to these communications.

License Verification

License key validation is performed locally using cryptographic verification. No internet connection is required for license validation after initial activation.

Permissions

The Accessibility, Screen Recording, and Input Monitoring permissions are used exclusively for RPA automation operations initiated by you. No data captured through these permissions is transmitted outside your Mac.
